|            |                    |                   |                    |

|------------|--------------------|-------------------|--------------------|
|            |                    |                   |                    |
|Hoisting    | Hosted at top Can  | Hosted at top can | Hosted at top can  |      
|            |   be used before   | not be used before| not be used before |
